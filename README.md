# 📈 Прогноз направления и волатильности акций с использованием гибридных моделей

**Дипломная работа**  
**Тема:** Разработка программы для прогнозирования направления и волатильности цен акций и фондовых индексов с использованием гибридных моделей

### Основные возможности
- Гибридные ML-модели (LogisticRegression + HistGradientBoosting + XGBoost + LightGBM + VotingClassifier)
- Прогноз на 5 дней вперёд (направление + волатильность)
- SHAP-интерпретация (почему модель решила именно так)
- Risk Management с расчётом VaR, позиции и рекомендациями
- Streamlit веб-интерфейс (RU/EN)
- Автоматическое обновление данных из yfinance + PostgreSQL

### Технологический стек
- **Backend**: Python 3.13, scikit-learn, XGBoost, LightGBM, SQLAlchemy
- **Данные**: yfinance + PostgreSQL
- **Frontend**: Streamlit + Plotly
- **Интерпретируемость**: SHAP
- **Контейнеризация**: Docker Compose

### Как запустить проект

```bash
# 1. Запуск базы данных
docker-compose up -d

# 2. Установка зависимостей
pip install -r requirements.txt

# 3. Полное обновление данных и моделей
python -m jobs.ingest_prices
python -m jobs.build_features
env ACTION=infer TASK=direction python -m jobs.train_baseline
env ACTION=infer TASK=volatility python -m jobs.train_baseline

# 4. Запуск сайта
streamlit run apps/web/main.py

# 📈 Прогноз направления и волатильности акций и фондовых индексов  
**Гибридные модели машинного обучения**

**Дипломная работа**  
**Тема:** Разработка программы для прогнозирования направления и волатильности цен акций и фондовых индексов с использованием гибридных моделей

## 🎯 Цель проекта

Разработать **полноценную производственную систему** прогнозирования финансовых инструментов, которая:
- Предсказывает **направление движения** цены (рост/падение) на 5 дней вперёд
- Предсказывает **ожидаемую волатильность** на 5 дней вперёд
- Даёт **чёткую торговую рекомендацию** («Покупать» / «Задуматься о покупке» / «Не покупать»)
- Объясняет своё решение с помощью **SHAP-интерпретации**
- Учитывает **риск-менеджмент** (VaR, размер позиции, уровень риска)

## 🚀 Основные возможности

- **Гибридная модель** высшего уровня (Soft Voting + XGBoost + LightGBM + HistGradientBoosting + Logistic Regression)
- Прогноз на **5 дней вперёд** (направление + волатильность)
- Реальная **SHAP-интерпретация** (почему модель приняла такое решение)
- Система **Risk Management** с расчётом VaR, рекомендуемой позиции и уровнем уверенности
- Красивый **веб-интерфейс** на Streamlit (поддержка русского и английского языков)
- Автоматическое обновление данных из `yfinance`
- Хранение данных в **PostgreSQL** (Docker)
- Полностью воспроизводимый пайплайн

## 🧠 Что такое гибридная модель в этом проекте

**Гибридная модель** — это комбинация нескольких разнородных алгоритмов, которые работают вместе.

В проекте используется **Soft Voting Ensemble** (мягкое голосование):

- **Logistic Regression** — ловит линейные зависимости
- **HistGradientBoosting** — мощный градиентный бустинг
- **XGBoost** — один из самых сильных бустингов
- **LightGBM** — быстрый и эффективный градиентный бустинг

**HYBRID_VOTING** (главная модель) усредняет вероятности от всех четырёх моделей.  
Это позволяет компенсировать слабые стороны каждой отдельной модели и получать более стабильные и точные прогнозы.

## 📊 Как работает вся система (пайплайн)

1. **ingest_prices.py** → скачивает свежие данные из yfinance
2. **build_features.py** → создаёт технические и макро-признаки (без утечки будущего)
3. **train_baseline.py** → проводит walk-forward валидацию и выбирает лучшую модель для каждого символа
4. **predict.py** → делает реальное предсказание + SHAP + Risk Management
5. **Streamlit сайт** → отображает результат пользователю
6. **Telegram-бот** (в разработке) → будет присылать ежедневные/еженедельные отчёты
